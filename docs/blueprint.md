# **App Name**: EpicBraids Online

## Core Features:

- Homepage Showcase: Display curated bracelets and keychains, plus a portal to customization.
- Product Listings: Display available bracelets and keychains.
- Product Detail Pages: Showcase products with images, descriptions, material information, and sizing for bracelets.
- Customization Interface: Enable users to select bracelet styles, colors, and sizes.
- Cart Management: Allow users to view, modify, and finalize their selected items.
- Order Processing: Collect customer details and finalize the order.
- AI-Powered Style Recommendation Tool: The app uses generative AI to offer a visual bracelet recommendation based on user color preferences.

## Style Guidelines:

- Primary color: Soft beige (#F5F5DC) to convey a handcrafted feel.
- Background color: Very light beige (#FAFAF5) provides a subtle, neutral backdrop.
- Accent color: Muted gold (#B8860B) adds a premium touch and highlights interactive elements.
- Body font: 'PT Sans' a humanist sans-serif for clear and accessible body text.
- Headline font: 'Belleza', a humanist sans-serif, to create a unique visual impact for headlines.
- Use minimalistic line icons to represent categories and actions.
- Mobile-first approach with a clean, grid-based structure for a premium feel.
- Subtle hover animations and smooth transitions for an enhanced user experience.
